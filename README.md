# Brasil API

Cliente para a API do BrasilAPI (para o Rust).

</div>

# Features
 - [x] **CEP (Zip code)**
 - [ ] **DDD**
 - [ ] **Bank**
 - [ ] **CNPJ**
 - [ ] **IBGE**
 - [ ] **Feriados Nacionais**
 - [ ] **Tabela FIPE**
 - [ ] **ISBN**
 - [ ] **Registros de domínios br**
 - [ ] **Taxas**


# Instalação
Adicione a seguinte linha ao seu `Cargo.toml`:

```toml
[dependencies]
brasilapi = "0.1.0"
```
