Uma [lib](https://crates.io/crates/brasilapi) para a API do [BrasilAPI](https://github.com/BrasilAPI/BrasilAPI) (para o Rust)

# Features
 - [x] **CEP (Zip code)**
 - [x] **DDD**
 - [ ] **Bank**
 - [ ] **CNPJ**
 - [ ] **IBGE**
 - [ ] **Feriados Nacionais**
 - [ ] **Tabela FIPE**
 - [ ] **ISBN**
 - [ ] **Registros de domínios br**
 - [ ] **Taxas**


# Instalação
Adicione a seguinte linha ao seu `Cargo.toml`:

```toml
[dependencies]
brasilapi = "0.2.0"
```
